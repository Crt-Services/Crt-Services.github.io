OMNI Operator/Technician Training

Details:

This training is a Basic Technician Training course for the OMNI 3000/6000 systems. This course will teach fundamental knowledge of the Operator and Technician class, along with experience in flow computers and measurement principles.

For operators, topics include Omni 3000 vs 6000 Hardware, Front Panel Keypad Navigation, Checking Meter Run Data, Batch Control, Viewing & Acknowledging Alarms, and Using Default and/or Custom Report Templates.

For technicians, topics include Configuration I/O Channels, Back panel I/O Channel connections, Wiring Various Devices, Advanced Front Panel Keypresses, OMNICOM Software, Executing a Prove Sequence, Transducer Fail Strategy & Overrides, Report Generation & Retrieval, Basic & Global Configuration Settings, Accessing Custom User Displays & Context Sensitive Help, Basic Serial & Ethernet Communications and much more.
