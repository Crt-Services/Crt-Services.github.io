ROC800 Basic Training Operations & Configuration

Details:

This course covers operations and configuration for the ROC800 system.

Topics:

  • Hardware & Software Overview: ROC800 Hardware Modules, ROCLINK800 Software, TLP Database Points Structure
  • Application Setup: License Key Administrator, Liquid vs Gas Applications, Using .tar Files to Enable Application Features
  • Configuration: Security, Setting Up Products, I/O Channels - Configuration, Assignments & Calibration
  • Logic & Programming: FST (Function Sequence Table) Commands, Soft Points (Scratchpads), Modbus Map Creation
  • Visualization & Reporting: User Displays, Historical Data Archiving, Report Templates, Modbus Communication, CFX File Creation
  • Advanced Features: Proving, Batching in Liquid Applications, Saving Configuration to Cold Boot Memory
  • Monitoring & Notifications: Alarms & Events
